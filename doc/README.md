# 職人Beauty（Shokunin Beauty）

建築職人・修繕リフォーム・エアコン取り付けなどの職人と発注者をつなぐ予約プラットフォーム

## プロジェクト概要

「職人Beauty」は、ホットペッパービューティーの職人版として、エアコン取り付け・水回り修繕・電気工事などの専門技術を持つ職人と、そのサービスを必要とする発注者をマッチングするWebプラットフォームです。

## ビジョン

> 「信頼できる職人に、簡単に依頼できる世界を」

従来、職人への依頼は知人の紹介や、どこに頼めばいいかわからないという課題がありました。本サービスでは、職人のプロフィール・評価・空き状況を可視化し、発注者が安心して予約できる仕組みを提供します。

## ターゲットユーザー

### 発注者（顧客）

- エアコン取り付けを依頼したい個人・法人
- 水回りの修繕を依頼したい住宅オーナー
- 電気工事を依頼したい店舗・オフィス
- リフォーム・修繕を検討している方

### 職人（事業者）

- 一人親方として活動する職人
- 小規模事業者
- 副業として技術を提供したい方
- 新規顧客を獲得したい事業者

## 解決する課題

| 発注者の課題 | 職人の課題 |
|-------------|-----------|
| どこに頼めばいいかわからない | 新規顧客の獲得が難しい |
| 料金相場がわからない | 空き時間を有効活用したい |
| 信頼できる職人か不安 | 自分の技術・実績をアピールしたい |
| 予約の手続きが面倒 | 予約管理が煩雑 |

## 競合との差別化

| 特徴 | 職人Beauty | 従来の方法 |
|------|-----------|-----------|
| 職人検索 | エリア・サービス・評価で簡単検索 | 知人紹介・チラシ |
| 料金 | 事前に料金目安がわかる | 見積もり依頼が必要 |
| 予約 | オンラインで即時予約 | 電話・メールで調整 |
| 評価 | レビュー・評価が見える | 口コミに依存 |
| 空き状況 | リアルタイムで確認可能 | 問い合わせが必要 |

## MVP（Minimum Viable Product）

最初のリリースでは、以下の機能に絞って開発します：

1. **職人検索・一覧表示** - エリアとサービスで職人を探せる
2. **職人プロフィール** - 経歴・サービス・料金・評価が見れる
3. **予約リクエスト** - 空き日時を選んで予約できる
4. **レビュー機能** - 作業完了後に評価を投稿できる

詳細は [requirements.md](./requirements.md) を参照してください。

## 技術スタック

- **フロントエンド**: Next.js 16 (App Router) + React 19 + Tailwind CSS 4
- **バックエンド**: Next.js API Routes + Supabase
- **データベース**: PostgreSQL (Supabase)
- **認証**: Supabase Auth
- **ホスティング**: Vercel

詳細は [architecture.md](./architecture.md) を参照してください。
