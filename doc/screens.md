## 画面遷移フロー

### 全体フロー

```mermaid
flowchart TB
    subgraph public [公開ページ]
        Top[トップページ]
        Search[職人検索]
        Detail[職人詳細]
        Login[ログイン]
        Signup[会員登録]
    end
    
    subgraph customer [発注者ページ]
        Book[予約フォーム]
        BookConfirm[予約確認]
        BookComplete[予約完了]
        MyPage[マイページ]
        MyBookings[予約一覧]
        BookingDetail[予約詳細]
        Review[レビュー投稿]
    end
    
    subgraph craftsman [職人ページ]
        Dashboard[ダッシュボード]
        CBookings[予約管理]
        CBookingDetail[予約詳細]
        Profile[プロフィール]
        Services[サービス管理]
        Availability[空き状況]
    end
    
    Top --> Search
    Top --> Login
    Top --> Signup
    Search --> Detail
    Detail --> Book
    Login --> MyPage
    Login --> Dashboard
    
    Book --> BookConfirm
    BookConfirm --> BookComplete
    BookComplete --> MyBookings
    MyPage --> MyBookings
    MyBookings --> BookingDetail
    BookingDetail --> Review
    
    Dashboard --> CBookings
    Dashboard --> Profile
    Dashboard --> Services
    Dashboard --> Availability
    CBookings --> CBookingDetail
```

### 予約フロー（発注者）

```mermaid
flowchart LR
    A[職人検索] --> B[職人詳細]
    B --> C[予約フォーム]
    C --> D[予約確認]
    D --> E[予約リクエスト送信]
    E --> F[予約完了画面]
    F --> G[予約一覧で確認]
```

### 予約承認フロー（職人）

```mermaid
flowchart LR
    A[予約リクエスト通知] --> B[予約管理画面]
    B --> C[予約詳細確認]
    C --> D{承認?}
    D -->|承認| E[予約確定]
    D -->|却下| F[予約却下]
    E --> G[作業実施]
    G --> H[作業完了報告]
```

## レスポンシブ対応

### ブレークポイント

| サイズ | 幅 | 対象 |
|--------|-----|------|
| sm | 640px〜 | スマートフォン横 |
| md | 768px〜 | タブレット |
| lg | 1024px〜 | デスクトップ |
| xl | 1280px〜 | 大画面 |

### モバイル対応方針

- **ナビゲーション**: ハンバーガーメニュー
- **検索**: 絞り込みはモーダル表示
- **職人カード**: 1列表示
- **フォーム**: フルワイド入力欄
- **サイドメニュー**: 下部タブナビゲーション（職人画面）
